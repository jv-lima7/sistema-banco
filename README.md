# 🏦 Banco Master

Sistema bancário via terminal desenvolvido em Java, com funcionalidades de consulta de saldo, depósito e saque.

---

## 📋 Funcionalidades

- **Consultar saldo** — exibe o saldo atual da conta
- **Depositar** — adiciona um valor ao saldo, com validação de entrada
- **Sacar** — retira um valor do saldo, verificando se há saldo suficiente
- **Sair** — encerra o programa

---

## 🗂️ Estrutura do Projeto

```
sistemabanco/
├── SistemaBanco.java   # Classe principal (main) com menu interativo
└── Conta.java          # Model com as operações bancárias
```

---

## ▶️ Como executar

### Pré-requisitos

- Java JDK 8 ou superior instalado
- IntelliJ IDEA (ou outra IDE Java)

### Passos

1. Clone ou baixe o repositório
2. Abra o projeto no IntelliJ IDEA
3. Localize a classe `SistemaBanco.java`
4. Clique com o botão direito e selecione **Run 'SistemaBanco.main()'**

---

## 🖥️ Como usar o sistema

Ao iniciar o programa, o menu principal é exibido:

```
--------BANCO MASTER--------
1 - Consultar saldo
2 - Depositar
3 - Sacar
4 - Sair

Escolha uma opção:
```

- Escolha **1** para ver seu saldo atual
- Escolha **2** para depositar um valor
- Escolha **3** para sacar um valor
- Escolha **4** para encerrar o programa
